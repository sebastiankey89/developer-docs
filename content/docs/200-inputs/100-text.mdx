Add a _text_ input to allow users to customize headings, call to actions or use it as a search parameter for external APIs.

<Callout>**Known issue:** Special characters are not encoded when passed in through the URL template.</Callout>

## Multiline

Enable the the _text_ inputâ€™s multiline option to allow users to type multiple lines of text.

<Callout>**Known issue:** Newline characters are not passed in through the URL template, preventing you from easily rendering multiple lines.</Callout>

## Validation

The _text_ input supports the following validation options:

- **Required:** Prevent your app from rendering until an input value is provided.
- **Max Length:** Prevent users from typing too many characters.
